#################################
Xilinx QDMA DPDK Poll Mode Driver
#################################

The Xilinx PCI Express Multi Queue DMA (QDMA) IP provides high-performance
direct memory access (DMA) via PCI Express.

Xilinx provides a DPDK poll mode driver based on DPDK v23.11/v22.11/v21.11/v20.11
that runs on a PCI Express root port host PC to interact with the QDMA
endpoint IP via PCI Express.
